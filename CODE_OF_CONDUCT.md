# Expectations
Be professional and respectful in all Github and third-party forums.
# Unacceptable behavior
Hate speech, personal attacks, and bigotry will result in exclusion from the community.
# How to report violations
Report violations to a core maintainer via discord or email.
# Enforcement
Enforcement such as removal from the project will be done at any core maintainers disgression.
# Community values
This project values an open marketplace of diversity and ideas.  No idea is stupid and none should be treated as such.  This does not mean all feature requests will be completed, or all PR's will be accepted.  Feedback should be constructive and professional, personal attacks are not tolerated.
