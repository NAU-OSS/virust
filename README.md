# Vi in the Web
Virust is an implementation of the vi editor written in rust and compiles to webassembly to be used in websites.

# Features
Basic vi text editing functionalities including normal, insert, and command modes.  Many motions implemented for a familiar vi/vim experience on the web.

# Installation
You can build from source or install the library as a npm package in a NodeJS project with:

```npm install virust```

# Usage example
## Usage in a React project.
In your desired React component, set the displayed text to the wasmbinding getText() on reload.  The binding input() should be called to handle keyboard input when the vi component is focused.
